┌─ SETUP (One Time) ────────────────────────────────────────────┐
│                                                                │
│  1. Open TWO terminal windows                                 │
│                                                                │
│  2. Terminal 1 - Start Watch Mode:                            │
│     $ cd /path/to/website                                     │
│     $ ./build.sh watch                                        │
│                                                                │
│  3. Terminal 2 - Start Web Server:                            │
│     $ cd /path/to/website                                     │
│     $ python3 -m http.server 8000                             │
│                                                                │
│  4. Open browser: http://localhost:8000                       │
│                                                                │
└────────────────────────────────────────────────────────────────┘

┌─ ADDING A PROJECT ────────────────────────────────────────────┐
│                                                                │
│  1. Create new file:                                          │
│     $ touch projects/my-project.md                            │
│                                                                │
│  2. Add content with frontmatter:                             │
│     ---                                                       │
│     title: My Project                                         │
│     date: 2024-10-20                                          │
│     tags: [web, javascript]                                   │
│     github: https://github.com/user/repo                      │
│     ---                                                       │
│                                                                │
│     ## Your content here...                                   │
│                                                                │
│  3. Watch it appear automatically! ✨                          │
│     • Watch mode detects change (~500ms)                      │
│     • Rebuilds projects.json                                  │
│     • Browser refreshes list (~2 seconds)                     │
│                                                                │
└────────────────────────────────────────────────────────────────┘

┌─ COMMANDS ────────────────────────────────────────────────────┐
│                                                                │
│  Build once (manual):                                         │
│    $ ./build.sh                                               │
│                                                                │
│  Build and watch (auto):                                      │
│    $ ./build.sh watch                                         │
│                                                                │
│  Test the system:                                             │
│    $ ./test-auto-update.sh                                    │
│                                                                │
│  Stop watch mode:                                             │
│    Press Ctrl+C                                               │
│                                                                │
└────────────────────────────────────────────────────────────────┘

┌─ TROUBLESHOOTING ─────────────────────────────────────────────┐
│                                                                │
│  Projects not updating?                                       │
│    • Check watch mode is running (Terminal 1)                │
│    • Verify .md file has frontmatter                          │
│    • Try manual build: ./build.sh                             │
│                                                                │
│  Browser not refreshing?                                      │
│    • Hard refresh: Cmd+Shift+R (Mac) / Ctrl+Shift+R (Win)    │
│    • Check browser console (F12) for errors                  │
│    • Verify web server is running (Terminal 2)               │
│                                                                │
│  Build errors?                                                │
│    • Check Node.js is installed: node --version               │
│    • Verify frontmatter YAML syntax                           │
│    • Check file permissions: chmod +x *.sh                    │
│                                                                │
└────────────────────────────────────────────────────────────────┘

┌─ WHAT HAPPENS AUTOMATICALLY ──────────────────────────────────┐
│                                                                │
│  When you save a .md file in /projects/:                      │
│                                                                │
│  1. Watch script detects file change                          │
│  2. Waits 500ms for changes to settle (debounce)              │
│  3. Runs build-projects.js                                    │
│  4. Regenerates projects.json                                 │
│  5. Browser polls projects.json every 2 seconds               │
│  6. Detects change via last-modified header                   │
│  7. Reloads project list                                      │
│  8. Updates navigation menu                                   │
│  9. Your new project appears! 🎉                               │
│                                                                │
│  Total time: 2-3 seconds from save to visible                 │
│                                                                │
└────────────────────────────────────────────────────────────────┘
